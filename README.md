## Issue Tracker
Issue tracker website helps you to keep track of issues you have in your projects, the website is made using the following:
* React.js
* Node.js
* Express.js
* MySQL
* MVC architecture
* JWT authentication
* nodemailer
* bootstrap
* HTML & CSS
